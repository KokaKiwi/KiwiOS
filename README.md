# KiwiOS
A small Operating System built from scratch based on some tutorials I've read.

## Building

### Requirements
You need somes packages to build and run KiwiOS:

- gcc
- nasm
- grub-mkrescue
- qemu

### Build
In a shell, type:

```sh
autoconf && ./configure && make
```

### Run
In a shell, type:

```sh
make run-qemu
```
